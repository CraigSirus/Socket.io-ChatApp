# Socket.io-ChatApp

A Chat application using socket.io

### Run -

- npm install 
- node index.js for the server
- Browser - ```http://localhost:3000/```

### Functionality

- Ability to have multiple users access the chat
- Each user can chat 
- Chat field will update for every user

### Built with - 

- ```Html```
- ```Css```
- ```Javascript```
- ```Jquery```
- ```Socket.io```
